# NQ Replay Skill

## 목적
NQ / NASDAQ 100 차트의 실시간 시나리오 작성, 사후 복기, 패턴 DB 누적을 일관된 방식으로 수행한다.

## 기준
- 1시간 방향 → 15분 구조 → 5분 트리거 순서로만 판단한다.
- `rules/trading_manual_v4.md`가 최우선이다.
- 더블비, 패턴, 거래량은 보조 필터다.
- 5분 단독 신호로 진입 시나리오를 확정하지 않는다.

## 복기 절차
1. 사전 시나리오를 요약한다.
2. 실제 흐름과 매칭한다.
3. 맞은 조건 / 틀린 조건 / 전환 조건을 분리한다.
4. 다음 실전에서 쓸 수 있는 규칙으로 정리한다.
5. `data/pattern_index.csv`에 태그를 추가한다.

## 출력 형식
- 메인 패턴명
- 사전 구조
- 실제 진행
- 전환 신호
- 좋은 진입 포인트
- 실패 조건
- 실전 교훈
- 다음 실전 적용 규칙
