# Trading Skill: NASDAQ Futures Volume Filter v1

## 0. 목적

이 문서는 `trading_manual_v4.md`를 보조하는 **나스닥 선물 거래량 기반 보조 필터**이다.

핵심 목적은 다음과 같다.

- 5분 진입 트리거의 신뢰도를 높인다.
- 15분 실행 구간에서 진짜 돌파와 가짜 돌파를 구분한다.
- 지지/저항 구간에서 매수·매도 압력의 질을 확인한다.
- 거래량 없는 돌파, 거래량 없는 반등, 중간값 추격 진입을 줄인다.

단, 거래량은 방향을 정하는 도구가 아니다.

> 거래량은 1시간 방향을 바꾸지 않는다.  
> 거래량은 15분 실행 구간을 새로 만들지 않는다.  
> 거래량은 이미 v4 기준으로 정해진 자리에서 5분 진입을 승인하거나 보류하는 보조 필터다.

---

# 1. 기존 지침과의 우선순위

매매 판단 우선순위는 반드시 아래 순서를 따른다.

1. `trading_manual_v4.md`
   - 1시간 방향
   - 15분 구조
   - 5분 트리거

2. `trading_skill_double_bollinger_1h_v1.md`
   - 1시간 과열/과매도 보조 판단
   - 추격 진입 제한
   - 평균회귀 압력 확인

3. `trading_skill_pattern_filter_mtf_v1.md`
   - 15분 실행 구간 안에서 5분 패턴 트리거 정밀화

4. `trading_skill_volume_filter_nq_v1.md`
   - 5분 트리거 최종 승인 필터
   - VWAP / Volume Profile / RVOL / CVD 확인
   - 거래량 기반 진입 품질 평가

v4와 충돌할 경우 항상 v4를 우선한다.  
거래량 필터는 단독 진입 신호가 아니다.

---

# 2. 데이터 사용 원칙

## 2-1. 권장 데이터

나스닥 선물 분석에서는 가능하면 다음 데이터를 우선 사용한다.

- CME NQ 선물 실제 체결량
- CME MNQ 선물 실제 체결량
- 세션 VWAP
- 세션 Volume Profile
- 고정 구간 Volume Profile
- CVD
- Footprint / DOM

## 2-2. TradingView NAS100 CFD 사용 시 주의

TradingView에서 `NAS100`, `US100`, `US Nas 100` CFD 차트를 볼 경우 거래량은 실제 CME 체결량이 아니라 **tick volume 또는 브로커 제공 거래량**일 수 있다.

이 경우 거래량 필터는 다음처럼 낮춰서 적용한다.

- 절대 거래량 수치보다 상대 거래량 증가 여부를 본다.
- RVOL은 방향 신호가 아니라 참여도 확인용으로만 쓴다.
- CVD가 없으면 가격 구조와 거래량 막대만으로 보조 판단한다.
- 실제 NQ/MNQ 데이터가 없으면 거래량 필터 점수는 최대 B급까지만 반영한다.

# 3. 핵심 도구 우선순위

거래량 보조 필터의 우선순위는 다음과 같다.

1. 세션 VWAP
2. 세션 / 고정 Volume Profile
   - VPOC
   - VAH
   - VAL
   - HVN
   - LVN
3. RVOL(20)
4. CVD
5. Footprint / DOM

OBV, PVT, Volume RSI는 보조 확인용으로만 사용한다.  
5분 직접 진입 필터로는 VWAP / VP / RVOL / CVD를 우선한다.

# 4. 세션 기준

## 4-1. RTH 우선

실전 진입 필터는 가능하면 미국 정규장 RTH를 우선한다.

- RTH: 09:30~16:00 ET
- 한국 시간: 22:30~05:00 또는 23:30~06:00, 서머타임 여부 확인

RTH 거래량은 신뢰도가 높고, VWAP / VPOC / VAH / VAL의 의미가 강하다.

## 4-2. ETH 주의

프리마켓, 애프터마켓, Globex 저유동성 시간에는 거래량 필터 신뢰도를 낮춘다.

ETH에서는 다음을 주의한다.

- 작은 거래량에도 RVOL이 과대평가될 수 있다.
- 얇은 호가로 가짜 돌파가 자주 발생한다.
- VWAP 리클레임이 나와도 지속성이 약할 수 있다.
- 진입은 가능하지만 등급을 한 단계 낮춘다.

---

# 5. 거래량 필터의 역할

거래량 필터는 세 가지 역할만 한다.

## 5-1. 승인

v4 방향, 15분 구간, 5분 트리거가 맞고 거래량도 동행하면 진입 승인.

예:

- 1시간 상승 우세
- 15분 지지구간
- 5분 저점 휩쏘 후 회복
- RVOL 상승
- VWAP 회복
- CVD 저점 상승

→ 롱 승인

## 5-2. 보류

가격 트리거는 나왔지만 거래량이 약하거나 VWAP/VPOC 기준이 애매하면 보류.

예:

- 5분 양봉 전환은 나왔지만 RVOL 0.8 이하
- VWAP 아래
- VPOC 아래
- CVD 동행 없음

→ 추격 금지, 재확인 대기

## 5-3. 경고

가격은 좋아 보이지만 거래량이 반대 방향을 말하면 경고.

예:

- 가격은 고점 돌파
- RVOL은 높음
- 그런데 CVD는 고점 하락
- VAH 위 안착 실패

→ 돌파 실패 가능성, 추격 롱 금지

---

# 6. RVOL 기준

RVOL은 현재 거래량을 최근 평균 거래량과 비교하는 상대 거래량이다.

기본 기준은 20봉 평균을 사용한다.

```text
RVOL = 현재 거래량 / 최근 20봉 평균 거래량
```

## 6-1. RVOL 해석

| RVOL | 해석 | 대응 |
|---|---|---|
| 0.8 미만 | 참여 부족 | 진입 보류 |
| 0.8~1.19 | 보통 | 가격 구조 우선 |
| 1.2 이상 | 의미 있는 참여 | 지지/회복 트리거 승인 가능 |
| 1.5 이상 | 강한 참여 | 이탈/붕괴/거절 신뢰도 상승 |
| 1.8 이상 | 매우 강한 참여 | 돌파 지속 가능성 상승 |
| 2.5 이상 | 클라이맥스 가능 | 추격 금지, 리테스트 대기 |

## 6-2. RVOL 금지 해석

- RVOL이 높다고 무조건 방향이 강한 것은 아니다.
- 고거래량 윗꼬리는 매수세가 아니라 매수 실패일 수 있다.
- 고거래량 아랫꼬리는 매도세가 아니라 매도 흡수일 수 있다.
- RVOL은 반드시 캔들 종가, VWAP, VPOC, CVD와 함께 해석한다.

---

# 7. VWAP 필터

VWAP은 세션의 거래량 가중 평균 가격이다.  
나스닥 선물 단타에서는 당일 공정가 기준으로 사용한다.

## 7-1. 기본 해석

| 가격 위치 | 해석 |
|---|---|
| 가격이 VWAP 위 | 당일 매수 우위 |
| 가격이 VWAP 아래 | 당일 매도 우위 |
| VWAP 근처 횡보 | 회전 / 노이즈 |
| VWAP 회복 | 롱 트리거 품질 상승 |
| VWAP 이탈 | 숏 트리거 품질 상승 |

## 7-2. 롱 승인 조건

다음 조건 중 다수가 맞으면 롱 승인 가능성이 높다.

- 15분 지지구간에서 반등
- 5분 저점 휩쏘 후 회복
- 5분 종가가 VWAP 위로 회복
- RVOL 1.2 이상
- VPOC 또는 VAL 위 종가
- CVD higher low

## 7-3. 숏 승인 조건

다음 조건 중 다수가 맞으면 숏 승인 가능성이 높다.

- 15분 저항구간에서 실패
- 5분 고점 휩쏘 후 실패
- 5분 종가가 VWAP 아래로 이탈
- RVOL 1.2~1.5 이상
- VPOC 또는 VAH 아래 종가
- CVD lower high 또는 저점 갱신

## 7-4. VWAP 근처 금지 규칙

VWAP을 위아래로 계속 넘나드는 구간에서는 진입하지 않는다.

특히 다음 상황은 관망한다.

- 가격이 VWAP ±0.1~0.2% 안에서 횡보
- VPOC도 근처에 있음
- 5분 패턴 미완성
- RVOL 1.0 이하
- 15분 방향 불명확

---

# 8. Volume Profile 필터

Volume Profile은 가격대별 거래량 분포를 보여준다.

## 8-1. 주요 용어

| 용어 | 의미 | 실전 해석 |
|---|---|---|
| VPOC | 거래량이 가장 많이 쌓인 가격 | 자석 / 공정가 / 회전 중심 |
| VAH | 가치영역 상단 | 상단 저항 또는 돌파 기준 |
| VAL | 가치영역 하단 | 하단 지지 또는 이탈 기준 |
| HVN | 고거래량 노드 | 머무는 구간 / 회전 구간 |
| LVN | 저거래량 노드 | 빠르게 통과할 수 있는 구간 |

## 8-2. 롱 품질 상승

다음 상황은 롱 품질이 올라간다.

- 1시간 상승 우세
- 15분 VAL / VPOC / 상승 FVG 지지
- 5분에서 VAL 아래 휩쏘 후 회복
- VPOC 위 5분 종가
- RVOL 1.2 이상
- CVD가 저점 상승

## 8-3. 숏 품질 상승

다음 상황은 숏 품질이 올라간다.

- 1시간 하락 우세
- 15분 VAH / VPOC / 하락 FVG 저항
- 5분에서 VAH 위 휩쏘 후 실패
- VPOC 아래 5분 종가
- RVOL 1.2~1.5 이상
- CVD가 고점 하락

## 8-4. LVN 목표 원칙

LVN은 가격이 빠르게 이동할 수 있는 공백 구간이다.

- 롱 돌파 후 위쪽 LVN이 열려 있으면 목표 확장 가능
- 숏 이탈 후 아래쪽 LVN이 열려 있으면 목표 확장 가능
- LVN 통과 후 다음 HVN / VAH / VAL에서 부분청산 검토

---

# 9. CVD 필터

CVD는 매수 체결량과 매도 체결량의 누적 차이를 보여준다.

## 9-1. 기본 해석

| CVD 상태 | 의미 |
|---|---|
| 가격 상승 + CVD 상승 | 매수 동행 |
| 가격 하락 + CVD 하락 | 매도 동행 |
| 가격 상승 + CVD 하락 | 상승 약화 / 흡수 가능성 |
| 가격 하락 + CVD 상승 | 하락 약화 / 흡수 가능성 |

## 9-2. 롱에 유리한 CVD

- 가격은 저점을 살짝 깼지만 CVD는 저점을 높임
- 지지구간에서 CVD가 더 이상 하락하지 않음
- VWAP 회복 시 CVD도 동반 상승
- 매도 거래량이 커졌는데 가격이 더 내려가지 못함

→ 매도 흡수 / 롱 후보

## 9-3. 숏에 유리한 CVD

- 가격은 고점을 살짝 넘겼지만 CVD는 고점을 낮춤
- 저항구간에서 CVD가 더 이상 상승하지 않음
- VWAP 이탈 시 CVD도 동반 하락
- 매수 거래량이 커졌는데 가격이 더 올라가지 못함

→ 매수 흡수 / 숏 후보

## 9-4. CVD가 없을 때

CVD가 없으면 거래량 필터를 다음으로 대체한다.

- 캔들 종가 위치
- 윗꼬리 / 아랫꼬리
- RVOL
- VWAP 회복 / 이탈
- VPOC 위아래 종가
- 5분 패턴 완성 여부

CVD가 없으면 거래량 점수에서 1점을 감점한다.

# 10. 핵심 매매 모델

## 모델 A. 지지 롱

### 조건

- 1시간 상승 우세 또는 최소 중립
- 15분 지지구간 도착
  - VAL
  - VPOC
  - 상승 FVG
  - EMA200
  - HL OTT 하단
- 5분 저점 휩쏘 후 회복 또는 쌍바닥 넥라인 돌파
- RVOL 1.2 이상
- VWAP 재상향 또는 VWAP 위 유지
- VPOC / VAL 위 5분 종가
- CVD higher low

### 진입

- 5분 리클레임 캔들 종가 후
- 또는 리클레임 구간 첫 재지지 후
- 더 보수적으로는 VWAP / VPOC 위 재지지 확인 후

### 손절

- 휩쏘 저점 아래
- VAL 아래 5분 종가
- VWAP 회복 실패 후 재이탈

### 목표

- 1차: VWAP
- 2차: VPOC
- 3차: VAH
- 확장: 위쪽 LVN 통과 후 다음 HVN / 1시간 저항

### 등급

- v4 롱 7점 이상 + 거래량 3점 이상 → A급 롱
- v4 롱 5~6점 + 거래량 2점 이상 → B급 롱
- 거래량 1점 이하 → 진입 보류

---

## 모델 B. 붕괴 숏

### 조건

- 1시간 하락 우세 또는 1시간 상승 안의 조정 국면
- 15분 지지구간 이탈
  - VAL 이탈
  - VPOC 이탈
  - 상승 FVG 실패
  - EMA200 이탈
- 이탈봉 RVOL 1.5 이상
- 가격이 VWAP 아래
- 가격이 VPOC 아래
- CVD 저점 갱신
- 되돌림에서 이탈한 구간 재저항

### 진입

- 5분 이탈봉 추격 금지
- 이탈 후 1차 되돌림에서 VAL / VPOC / VWAP 재저항 확인 후
- 5분 음봉 전환 또는 직전 저점 재이탈 시

### 손절

- 재저항 구간 위 회복
- VWAP 위 5분 종가
- VPOC 위 재안착

### 목표

- 1차: 직전 5분 저점
- 2차: 다음 VAL / LVN
- 3차: 1시간 지지 / 하단 FVG / 하단 BB

### 등급

- 1시간 하락 우세 + 15분 이탈 + 거래량 3점 이상 → A급 숏
- 1시간 상승 우세 중 조정 숏이면 → B급 이하, 짧게 대응

---

## 모델 C. 거절 숏

### 조건

- 15분 저항구간 도착
  - VAH
  - VPOC
  - 하락 FVG
  - EMA200
  - HL OTT 상단
- 저항 터치 시 RVOL 1.5 이상
- 고거래량이 나왔지만 가격이 위에 안착 실패
- 고점 위 꼬리 후 저항 아래 종가
- CVD가 가격 고점과 동행하지 않음
- 다음 5분 저점 이탈

### 진입

- 고점 휩쏘 후 실패 확인
- 다음 5분 저점 이탈
- 또는 저항 재테스트 실패 후 음봉 전환

### 손절

- 휩쏘 고점 위
- VAH 위 5분 종가
- VWAP / VPOC 위 재안착

### 목표

- 1차: VWAP
- 2차: VPOC
- 3차: VAL
- 확장: 아래쪽 LVN

### 등급

- 1시간 하락 우세와 맞으면 A급 숏
- 1시간 상승 우세에서 나오면 역추세 단타 숏

---

## 모델 D. 돌파 롱

### 조건

- 1시간 상승 우세 또는 강한 전환 구조
- 15분 박스 상단 / VAH / VPOC 돌파
- 돌파봉 RVOL 1.8 이상
- 가격이 VWAP 위
- CVD가 고점 갱신 또는 강한 동행
- 돌파 위쪽에 LVN 공간 존재
- 첫 리테스트에서 상단 재지지

### 진입

- 돌파 직후 추격보다 첫 리테스트 우선
- 박스 상단 / VAH / VWAP / VPOC 재지지 확인 후
- 5분 양봉 전환 또는 직전 고점 재돌파 시

### 손절

- 돌파 상단 재이탈
- VWAP 아래 재이탈
- 돌파봉 저점 이탈

### 목표

- 1차: 직전 고점
- 2차: 다음 HVN
- 3차: 1시간 저항 / 상단 FVG / 상단 BB
- LVN이 열려 있으면 목표 확장 가능

### 등급

- v4 롱 7점 이상 + 거래량 3점 이상 → A급 돌파 롱
- RVOL은 높지만 CVD가 동행하지 않으면 가짜 돌파 경고

---

## 모델 E. 돌파 실패 숏

### 조건

- 가격이 15분 VAH / 박스 상단 / 고점 위로 돌파
- 돌파 시 RVOL 1.5 이상
- 하지만 5분 종가가 돌파 구간 아래로 복귀
- CVD가 고점 갱신 실패
- VWAP 또는 VPOC 위 안착 실패
- 다음 5분 저점 이탈

### 진입

- 돌파 실패 후 박스 내부 재진입
- 재진입 후 5분 저점 이탈
- 또는 돌파 구간 되돌림 재저항

### 손절

- 휩쏘 고점 위
- 돌파 구간 위 재안착
- VAH 위 5분 종가

### 목표

- 1차: VWAP
- 2차: VPOC
- 3차: 박스 하단 / VAL

### 등급

- 1시간이 하락 또는 중립이면 B~A급
- 1시간 강한 상승이면 짧은 역추세 단타만 가능

# 11. 거래량 점수표

거래량 필터는 v4 점수표에 보조 점수로만 반영한다.

## 11-1. 롱 거래량 점수

각 항목 1점.

- 15분 지지구간에서 발생
- RVOL 1.2 이상
- VWAP 위 회복 또는 VWAP 위 유지
- VPOC / VAL 위 5분 종가
- CVD higher low 또는 매도 흡수
- 위쪽 목표까지 LVN 공간 존재

### 해석

- 4점 이상: 롱 거래량 승인
- 2~3점: 조건부 승인
- 1점 이하: 진입 보류

## 11-2. 숏 거래량 점수

각 항목 1점.

- 15분 저항구간 또는 지지 이탈 구간에서 발생
- RVOL 1.2 이상, 붕괴는 1.5 이상
- VWAP 아래 이탈 또는 VWAP 아래 유지
- VPOC / VAH 아래 5분 종가
- CVD lower high 또는 매수 흡수
- 아래쪽 목표까지 LVN 공간 존재

### 해석

- 4점 이상: 숏 거래량 승인
- 2~3점: 조건부 승인
- 1점 이하: 진입 보류

# 12. v4 점수와 결합

## 12-1. A급 진입

다음 조건을 모두 만족할 때 A급으로 본다.

- v4 점수 7점 이상
- 15분 실행 구간 명확
- 5분 트리거 완성
- 패턴 필터 조건 충족
- 거래량 점수 4점 이상

## 12-2. B급 진입

다음 조건이면 B급으로 본다.

- v4 점수 5~6점
- 15분 실행 구간은 있음
- 5분 트리거는 완성
- 거래량 점수 2~3점

B급은 비중 축소 또는 짧은 목표 대응.

## 12-3. 관망

다음 중 하나라도 해당하면 관망한다.

- v4 점수 4점 이하
- 15분 실행 구간 밖
- 5분 트리거 미완성
- 거래량 점수 1점 이하
- VWAP / VPOC 근처에서 횡보
- RVOL 0.8 미만
- 뉴스 직후 초고변동
- 롤오버 / 휴장 전후 / 저유동성 시간

# 13. 거래량 기반 금지 규칙

- 거래량만 보고 진입 금지
- RVOL 스파이크만 보고 추격 금지
- VWAP 위라고 무조건 롱 금지
- VWAP 아래라고 무조건 숏 금지
- VPOC 근처 횡보에서 방향 매매 금지
- 15분 구간 밖의 거래량 스파이크 무시
- CVD 다이버전스만 보고 선진입 금지
- 고거래량 장대봉 이후 첫 캔들 추격 금지
- RVOL 2.5 이상 클라이맥스에서는 리테스트 전 진입 금지
- ETH 저유동성 시간의 RVOL 신호 과신 금지
- 뉴스 직후 첫 5분봉 거래량 신호 과신 금지

# 14. 실전 적용 순서

거래량 필터는 반드시 아래 순서로 적용한다.

## Step 1. 1시간 방향 확인

v4 기준으로 1시간 방향을 먼저 판단한다.

- EMA200
- POC
- VAH / VAL
- FVG
- 더블비
- 채훈이격도

## Step 2. 15분 실행 구간 지정

15분에서 오늘의 실제 관찰 구간을 정한다.

- VAL
- VPOC
- VAH
- FVG
- EMA200
- HL OTT
- 이전 고점 / 저점

## Step 3. 5분 가격 트리거 확인

5분에서 가격 트리거를 먼저 확인한다.

- 저점 휩쏘 후 회복
- 고점 휩쏘 후 실패
- 박스 돌파 후 재지지
- 박스 이탈 후 재저항
- EMA200 재돌파 / 재이탈
- 5분 FVG 방어 / 저항

## Step 4. 거래량 필터 확인

가격 트리거 후 거래량을 확인한다.

- RVOL 기준 충족 여부
- VWAP 위아래 위치
- VPOC / VAH / VAL 기준 종가
- CVD 동행 또는 다이버전스
- LVN 목표 공간

## Step 5. 최종 승인

- 가격 트리거 + 거래량 승인 → 진입 가능
- 가격 트리거 + 거래량 조건부 → 비중 축소 또는 재확인
- 가격 트리거 + 거래량 부정 → 진입 보류

# 16. 한 줄 운영 원칙

> v4가 방향을 정하고, 15분이 자리를 정하고, 5분이 트리거를 만든다.  
> 패턴 필터가 트리거의 모양을 정밀화하고,  
> 거래량 필터는 그 트리거에 실제 참여가 붙었는지 최종 승인한다.  
> 거래량이 강해도 15분 실행 구간 밖이면 진입하지 않는다.
