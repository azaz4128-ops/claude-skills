# 선차트 시나리오 이미지 출력 지침

사용자가 **“선차트 그려줘”**, **“시나리오 선차트”**, **“차트에 시나리오 그려줘”**라고 요청하면, 반드시 아래 형식으로 이미지를 생성한다.

---

## 1. 기본 스타일

- 다크 테마 TradingView 스타일
- NASDAQ 100 / NAS100 / NQ 차트 분석용 인포그래픽 형식
- 사용자가 첨부한 차트 흐름을 기반으로 한다.
- 중앙에는 실제 차트 흐름을 배경처럼 배치하고, 그 위에 시나리오 선을 표시한다.
- 시나리오 선은 가격 흐름을 명확히 보여주는 꺾은선 형태로 그린다.
- 단순 예쁜 이미지가 아니라 실제 매매 판단용 시나리오 맵처럼 만든다.
- 텍스트는 한국어로 작성한다.

---

## 2. 상단 구성

상단에는 3개의 요약 박스를 배치한다.

### 2-1. 1시간 방향

포함 내용:

- 큰 방향
- 주요 저항
- 주요 지지
- 더블비 / 과열 / 평균회귀 여부

### 2-2. 15분 구조

포함 내용:

- POC / VAH / VAL
- FVG
- 박스권 / 이탈 / 재저항 구간
- 숏 우세 또는 롱 우세 판단

### 2-3. 5분 현재 상황

포함 내용:

- 현재 트리거 상태
- 급락 / 반등 / 횡보 여부
- RSI / 거래량 / VWAP / VPOC 보조 판단
- 핵심 관찰 가격

---

## 3. 우측 상단 시나리오 범례

우측 상단에는 시나리오 범례를 둔다.

- 빨강: 시나리오 1 — 메인 숏
- 파랑: 시나리오 2 — 붕괴 숏
- 초록: 시나리오 3 — 역추세 반등 롱

---

## 4. 중앙 메인 차트 구성

중앙에는 가격 차트와 주요 레벨을 표시한다.

반드시 포함할 요소:

- 현재가 라인
- 주요 저항선
- 주요 지지선
- POC / 이전 지지 / 재저항 구간
- 최근 이탈 구간
- 급락 또는 거래량 증가 구간
- 시나리오별 예상 경로 선

### 4-1. 색상 규칙

- 메인 숏 경로: 빨간색 선
- 붕괴 숏 경로: 파란색 선
- 역추세 반등 롱 경로: 초록색 선

### 4-2. 경로 표시 규칙

- 각 경로에는 ① ② ③ 번호를 붙인다.
- 번호는 진입 → 확인 → 목표 흐름이 보이게 배치한다.
- 시나리오 선은 실제 가격 레벨과 최대한 맞춰 그린다.
- 현재 가격에서 바로 진입이 불리한 경우, 반드시 반등 대기 / 리테스트 대기 흐름으로 표현한다.

---

## 5. 하단 시나리오 박스 구성

하단에는 3개의 시나리오 박스를 배치한다.

### 5-1. 시나리오 1: 반등 후 재저항 숏

포함 내용:

- 진입 조건
- 손절
- 목표
- 확률 / 등급
- 위험도
- 수익성

### 5-2. 시나리오 2: 저점 이탈 후 붕괴 숏

포함 내용:

- 진입 조건
- 손절
- 목표
- 확률 / 등급
- 위험도
- 수익성

### 5-3. 시나리오 3: 역추세 반등 롱

포함 내용:

- 진입 조건
- 손절
- 목표
- 확률 / 등급
- 위험도
- 수익성

---

## 6. 최하단 요약 박스 구성

최하단에는 3개의 요약 박스를 둔다.

### 6-1. 현재 전략 요약

포함 내용:

- 추격 금지 여부
- 기다릴 구간
- 가장 유리한 시나리오

### 6-2. 핵심 가격 레벨

포함 내용:

- 저항
- 중간
- 지지

### 6-3. 주의 사항

포함 내용:

- 거래량 부족 시 관망
- VWAP / VPOC 근처 횡보 시 관망
- 주요 레벨 위 / 아래 안착 시 시나리오 실패 조건

---

## 7. 분석 기준

이미지에 들어가는 시나리오는 반드시 아래 순서로 판단한다.

1. 1시간 방향
2. 15분 구조
3. 5분 트리거

보조 판단은 아래 순서로만 적용한다.

1. 1시간 더블비
2. 15분 / 5분 패턴 필터
3. 거래량 필터

---

## 8. 출력 원칙

- 가격 레벨은 사용자가 첨부한 차트 분석 결과에 맞춰 표시한다.
- 차트 방향은 반드시 `trading_manual_v4.md` 기준의 분석 결과를 따른다.
- 더블비, 패턴, 거래량 필터는 보조 판단으로만 반영한다.
- 1시간 방향 → 15분 구조 → 5분 트리거 순서를 깨지 않는다.
- 억지 진입 시나리오는 만들지 않는다.
- 조건이 부족하면 “관망” 또는 “대기” 시나리오를 포함한다.
- 롱과 숏이 모두 가능해 보여도, 상위 프레임과 15분 구조상 더 유리한 방향을 메인 시나리오로 표시한다.

---

## 9. 고정 이미지 레이아웃 요약

사용자가 선차트를 요청하면 아래 레이아웃을 기본값으로 사용한다.

```text
[제목: NASDAQ 100 / NAS100 5분 차트 시나리오]

[1시간 방향 박스] [15분 구조 박스] [5분 현재 상황 박스] [시나리오 범례]

------------------------------------------------------------
|                                                          |
|                  중앙 메인 차트 영역                     |
|     실제 차트 흐름 + 저항/지지 + 시나리오 선              |
|                                                          |
------------------------------------------------------------

[시나리오 1 박스] [시나리오 2 박스] [시나리오 3 박스]

[현재 전략 요약] [핵심 가격 레벨] [주의 사항]
```

---

## 10. 한 줄 지침

> 사용자가 “선차트 그려줘”라고 요청하면, 단순한 선 그림이 아니라 **1시간 방향, 15분 구조, 5분 트리거를 반영한 다크 테마 TradingView 스타일 시나리오 맵**으로 만든다.
