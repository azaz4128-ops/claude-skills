# NQ Replay Skill

## 목적
NQ / NASDAQ 100 차트의 실시간 시나리오 작성, 사후 복기, 패턴 DB 누적을 일관된 방식으로 수행한다.

## 기준
- 1시간 방향 → 15분 구조 → 5분 트리거 순서로만 판단한다.
- `rules/trading_manual_v4.md`가 최우선이다.
- 더블비, 패턴, 거래량은 보조 필터다.
- 5분 단독 신호로 진입 시나리오를 확정하지 않는다.

## 차트 분석 답변 형식
사용자가 차트 이미지를 첨부하고 "차트 분석"을 요청하면 항상 아래 순서로 답한다.

1. 결론
   - 방향, 즉시 진입 가능 여부, 핵심 전략을 먼저 말한다.
   - "방향은 롱 우세, 타점은 눌림 대기"처럼 실전적으로 쓴다.

2. 1시간 방향
   - 상위 프레임 방향, 더블비, RSI, 과열/평균회귀, 주요 지지/저항을 정리한다.
   - 핵심 구간은 표로 쓴다.

3. 15분 구조
   - 현재 실행 구간, VAH/VAL/POC, 박스, FVG, 지지/저항을 정리한다.
   - 유지 조건과 깨지는 조건을 분리한다.

4. 5분 트리거
   - 트리거 완성 여부를 먼저 판단한다.
   - 박스 돌파/이탈, 재지지/재저항, 쌍바닥/쌍봉, 5분 종가 확인을 본다.
   - RSI, 거래량, CVD는 단독 진입 신호가 아니라 승인 필터로만 사용한다.

5. 매매 시나리오
   - 시나리오 1: 메인
   - 시나리오 2: 보조 또는 역추세 단타
   - 시나리오 3: 실패/전환
   - 각 시나리오에는 진입 구간, 트리거, 손절, 1차 목표, 2차 목표, 등급을 포함한다.

6. 최종 판단
   - 지금 할 행동을 관망/눌림 대기/돌파 후 눌림 확인/이탈 후 재저항 확인처럼 명확히 쓴다.
   - 추격 금지, 5분 종가 확인 전 트리거 미완성 같은 판단을 분명히 말한다.

## 복기 절차
1. 사전 시나리오를 요약한다.
2. 실제 흐름과 매칭한다.
3. 맞은 조건 / 틀린 조건 / 전환 조건을 분리한다.
4. 다음 실전에서 쓸 수 있는 규칙으로 정리한다.
5. `data/pattern_index.csv`에 태그를 추가한다.

## 출력 형식
- 메인 패턴명
- 사전 구조
- 실제 진행
- 전환 신호
- 좋은 진입 포인트
- 실패 조건
- 실전 교훈
- 다음 실전 적용 규칙
